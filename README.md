## Overview

Task Management Application built using Next.js and Express.js.

Users can:
- Add tasks
- View tasks
- Edit tasks
- Delete tasks
- Toggle task status

---

## Features

### Create Task
Create a task using title and description.

### Read Tasks
View all tasks.

### Update Task
Edit task details and toggle completion.

### Delete Task
Remove tasks.

### Responsive UI
Works across screen sizes.
